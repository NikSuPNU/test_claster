# Микросервисная архитектура системы сбора и обработки данных
Система включает в себя:
- **Airflow** для оркестрации задач
- **Spark** для аналитики данных (слияние и пересечение)
- **Kafka** для стриминга данных
- **S3** для хранения сырых данных в так и обработанных формате parquet для экономии места
- **PostgSQL** для хранения обработанных данных
